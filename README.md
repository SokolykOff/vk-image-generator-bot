# vk-image-generator-bot
VK-бот для генерации изображений по тексту с помощью нейросетей

## О проекте

Бот для ВКонтакте создаёт уникальные картинки по вашему тексту. Напишите, что хотите увидеть, и бот пришлёт **3 варианта**. Работает с нейросетями, понимает русский язык, кэширует результат.

## Примеры генераций

*Нажмите на картинку, чтобы увеличить*

| Ваш запрос | Результат |
|------------|-----------|
| «черепаха в скафандре сидит на Луне и смотрит на Землю, стиль футуризм, яркие звёздые» | 
| робот-пылесос в ковбойской шляпе гонится за котом, забавный мультяшный стиль|
| «киберпанк город ночью, неоновые вывески, летающие машины, аниме стиль» | 
| «кот-программист пьёт кофе» | 

## Возможности

- 🎨 Генерация **трёх** вариантов за раз
- ⚡ Кэширование – повторные запросы мгновенны
- 🌐 Автоматический перевод с русского на английский
- 🖼 Случайный аватар в разных стилях
- 📱 Простое меню с кнопками

## Технологии

- Python 3.14
- `vk_api`
- `aiohttp`, `requests`
- `deep-translator`
- `concurrent.futures` (параллельная генерация)

## Установка и запуск (для разработчиков)

1. Клонируйте репозиторий:  
   `git clone https://github.com/твой_логин/vk-image-generator-bot.git`
2. Установите зависимости:  
   `pip install -r requirements.txt`
3. Создайте файл `.env` в папке проекта и пропишите:
VK_TOKEN=токен_вашей_группы_вк
GROUP_ID=айди_группы

text
4. Запустите бота:  
`python vk_prompt_bot.py`
